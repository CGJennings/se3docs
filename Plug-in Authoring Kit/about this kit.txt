The purpose of this kit is to help you learn how to
write plug-ins for Strange Eons 3. You will also
want to consult the Developer Manual for additional
background and context on these examples.

Developer manual:
http://se3docs.cgjennings.ca/dm-index.html

JavaScript (.js) API docs:
http://se3docs.cgjennings.ca/assets/jsdoc/index.html

Java class API docs:
http://se3docs.cgjennings.ca/assets/javadoc/

---

You can reuse the example code and files in the
"Graphics Templates" folder in your own projects.
