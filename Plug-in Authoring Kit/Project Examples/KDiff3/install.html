KDiff3 Integration

Use KDiff3 to compare files and folders in a project.

To Compare Files or Folders

To make a comparison, select either 2 or 3 files or folders in the project. The selection must be either all files or all folders. When selecting files, only text files are supported (anything that can be edited with the code editor, such as script files, settings files, tile sets, HTML, etc.). Right click and choose Compare Files. A new KDiff3 window will open to display the comparison.

KDiff3 Configuration

This plug-in requires that KDiff3 be installed on your computer. The plug-in will look for KDiff3 in a platform-specific default location. If KDiff3 can't be found there, you will be prompted to paste in the full path to the binary executable.

Note: If for some reason you wish to change the KDiff3 path manually, you can do so by changing the value of the kdiff3-exe user setting.
