Zip Tools

Compress and decompress Zip archives in a project.

To Decompress Archives

Double-click a Zip archive to decompress it. All files in the archive will be decompressed into a folder with the same name as the archive.

To Compress Files

Select the files that you wish to compress, then right click and choose Compress. The selected files will be combined into a single archive. If the selected files are all in the same folder, the archive is created in that folder. Otherwise, the archive is placed in the project folder.
